纳米流式检测技术：从原理到生物医学应用

引言

流式细胞术（Flow cytometry）是现代生物医学研究中不可或缺的技术之一，已经在临床诊断、免疫学、肿瘤学等领域得到广泛应用。传统流式细胞术能够快速对单个细胞进行多参数分析，每秒可以检测数千到数万个细胞，极大地提高了细胞分析的效率。然而，传统流式细胞术的检测下限通常在~500-1000 nm左右，对于粒径更小的纳米颗粒（如细胞外囊泡、外泌体、病毒颗粒、纳米药物等），传统流式往往无能为力。

纳米流式检测技术（Nanoflow cytometry，也称为纳米流式细胞术）正是为了解决这一挑战而发展起来的新兴技术，能够对几十纳米到数百纳米的纳米颗粒进行高通量单颗粒检测和分析。近年来，随着细胞外囊泡、外泌体研究的热潮，以及纳米药物产业的快速发展，纳米流式技术得到了越来越多的关注，已经成为纳米生物医学研究领域的重要工具。

本文将系统介绍纳米流式检测技术的发展历史、基本原理、仪器构造，以及在生物医学领域的主要应用，并对未来发展方向进行展望。

一、发展历史与技术背景

1.1 传统流式细胞术的局限性

传统流式细胞术诞生于1960-1970年代，经过半个多世纪的发展，已经非常成熟。其检测原理是将细胞悬浮在液体中，通过聚焦的激光束，每个细胞穿过激光束时产生散射光和荧光信号，被检测器捕获从而得到细胞的大小、颗粒性以及荧光表达信息。

然而，传统流式的检测下限受限于光学系统和信噪比：

  • 光学衍射极限：根据瑞利判据，光学显微镜的分辨率极限约为λ/2，对于波长488 nm的激光，分辨率约为240 nm
  • 散射光强度：散射光强度与颗粒粒径的6次方成正比（瑞利散射），粒径减小一半，散射强度降低到1/64，信噪比急剧下降
  • 背景噪音：鞘液中的杂质颗粒会产生严重背景干扰
  • 浓度检测范围：难以同时检测低浓度大颗粒和高浓度小颗粒

因此，传统流式细胞术通常只能有效检测粒径 > 500-1000 nm的颗粒，对于300 nm以下的颗粒检测非常困难，而生物体系中许多重要颗粒恰恰落在这个范围：

颗粒类型 典型粒径范围 传统流式能否检测
常规哺乳动物细胞 10-30 μm ✅ 轻松检测
细菌 0.5-5 μm ⚠️ 大细菌可以，小细菌困难
病毒 20-300 nm ❌ 无法检测
细胞外囊泡/外泌体 30-200 nm ❌ 大部分无法检测
脂质纳米颗粒 50-200 nm ❌ 无法检测
高分子纳米药物 10-200 nm ❌ 无法检测

可以看到，许多重要的生物纳米颗粒都落在传统流式的”盲区”，这催生了对纳米流式检测技术的需求。

1.2 纳米流式的发展历程

纳米流式检测技术的发展经历了几个阶段：

  1. 1990年代-2000年代初：探索阶段，研究人员尝试改进传统流式的光学系统，尝试检测更小的颗粒，但灵敏度提升有限
  2. 2000年代中期-2010年代初：香山实验室等团队系统性地改进了纳米流式的光学设计、进样系统和信号处理方法，奠定了现代纳米流式的技术基础
  3. 2010年代中期至今：商业化成熟阶段，出现了商品化的纳米流式仪器，随着外泌体研究热潮，得到广泛应用

在中国，厦门大学-韩家淮院士/厦大团队在纳米流式领域做了大量开创性工作，开发了具有自主知识产权的纳米流式仪器，推动了该技术在中国的普及。

二、基本原理与核心技术

2.1 纳米颗粒检测的光学挑战

纳米颗粒检测面临的核心光学挑战是信噪比问题：

根据瑞利散射理论，散射光强度 ( I_s ) 与粒径 ( d ) 的关系为：

[ I_s \propto \frac{d^6}{\lambda^4} ]

也就是说：

  • 粒径从1000 nm降到100 nm，粒径缩小10倍，散射强度降低 ( 10^6 = 1,000,000 ) 倍
  • 这对检测灵敏度提出了极高要求

因此，纳米流式需要从光学系统设计、鞘液净化、信号放大、噪音抑制多个环节进行系统优化才能实现有效检测。

2.2 核心设计原理

现代纳米流式检测技术的核心设计包括以下几个方面：

1. 优化的光学聚焦系统

  • 采用更紧的激光聚焦，减小光斑尺寸，提高能量密度
  • 使用侧向散射和荧光收集的优化光路，提高收集效率
  • 高质量光学元件降低散射背景

2. 鞘液净化系统

  • 传统流式鞘液中含有大量纳米级杂质颗粒，会产生严重背景
  • 纳米流式需要对鞘液进行过滤净化，通常采用多级过滤（0.02 μm过滤）
  • 有些设计还采用了 degassing 减少气泡干扰

3. 高灵敏度检测器

  • 采用光电倍增管（PMT）代替雪崩光电二极管（APD），增益更高，噪音更低
  • 降低电子学噪音，提高弱信号检测能力

4. 先进信号处理

  • 采用脉冲高度分析区分真实纳米颗粒信号和背景噪音
  • 适合纳米颗粒浓度的计数算法

2.3 性能参数

评价纳米流式仪器性能的主要参数包括：

参数 含义 优秀纳米流式指标
检测下限 能够有效检测的最小粒径 ~20-40 nm

目前顶尖的纳米流式仪器可以稳定检测20 nm的聚苯乙烯纳米颗粒，对于生物颗粒可以检测~30-50 nm，基本覆盖了绝大多数生物医学感兴趣的纳米颗粒范围。

三、仪器构造与关键部件

3.1 整体构造

纳米流式仪器的基本构造和传统流式类似，但每个部件都有优化：

3.2 进样与聚焦系统

进样和聚焦系统是纳米流式的关键之一：

  • 剪切流聚焦：和传统流式一样，采用鞘液包裹样本流，将样本流聚焦到激光中心，保证每个颗粒依次穿过激光
  • 低流速：纳米颗粒浓度通常较高，需要较低流速保证计数准确
  • 无残留设计：避免样本交叉污染，对于临床样本尤其重要

3.3 激光系统

现代纳米流式通常采用多激光配置，和传统流式类似：

  • 常用激光器：405 nm、488 nm、561 nm、640 nm
  • 功率：几十毫瓦，足够激发荧光
  • 空间聚焦：将激光聚焦到~10 μm光斑，匹配纳米颗粒尺寸

3.4 信号检测系统

  • 侧向散射（SSC）：用于检测纳米颗粒本身的存在，提供粒径信息
  • 多个荧光通道：用于检测标记的荧光信号，获取分子表达信息
  • 光电倍增管（PMT）：每个通道一个PMT，提供高增益低噪音

3.5 数据分析系统

纳米流式输出的主要数据包括：

  • 颗粒浓度：单位体积内的颗粒数
  • 粒径分布：通过散射光强度估算粒径
  • 荧光强度分布：每个颗粒的荧光强度统计
  • 阳性颗粒比例：表达特定分子的颗粒比例

四、纳米流式在细胞外囊泡研究中的应用

细胞外囊泡（Extracellular vesicles, EVs）研究是目前纳米流式最活跃的应用领域，因为外泌体尺寸正好落在纳米流式的检测范围内。

4.1 外泌体/细胞外囊泡简介

细胞外囊泡是细胞分泌的纳米级膜囊泡，直径分布大约30-1000 nm，其中：

  • 外泌体（exosomes）：30-150 nm
  • 微囊泡（microvesicles）：100-1000 nm

EVs携带蛋白质、RNA、脂质等生物活性分子，参与细胞间通信，在肿瘤转移、免疫调节、诊断预后等方面有重要意义，是当前生物医学研究热点。

4.2 纳米流式相比传统方法的优势

EVs研究常用的表征方法有：

方法 能否计数 能否分析单个颗粒 能否多标记 通量 缺点
纳米颗粒跟踪分析（NTA） ✅ ✅ ❌ 低 无法做免疫分析
动态光散射（DLS） ❌ ❌ ❌ 极低 只能测平均粒径
透射电镜（TEM） ❌ ✅ ✅ 极低 无法定量
Western Blot ❌ ❌ ✅ 极低 整体分析，无法单细胞/单囊泡
纳米流式 ✅ ✅ ✅ 高 需要荧光标记

纳米流式是少数能够对单个EV进行高通量多参数免疫分析的方法，这是它最大的优势。

4.3 主要应用方向

1. EVs 计数和粒径分布分析

  • 可以快速得到样本中EVs的浓度和粒径分布
  • 比NTA通量高一个数量级以上
  • 可以处理大量样本，适合临床样本批量检测

2. 免疫表型分析

  • 可以用荧光标记抗体标记EVs表面特定蛋白
  • 纳米流式可以检测每个EVs的蛋白表达
  • 能够分析表达特定标志物的EVs比例
  • 可以发现肿瘤来源的EVs，用于癌症诊断

3. 尺寸-表达关联分析

  • 可以同时得到每个EV的粒径（散射强度）和荧光表达
  • 分析不同尺寸亚群的表达差异
  • 研究发现，小尺寸外泌体往往富集更多肿瘤相关标志物

4. 临床样本检测

  • 血液、唾液、尿液等体液中的EVs可以直接检测（适当稀释后）
  • 不需要超速离心纯化，减少纯化过程带来的偏差
  • 可以保留原始浓度信息，有利于定量诊断

案例研究：厦门大学团队利用纳米流式检测胰腺癌患者血浆中的外泌体，发现GPC1阳性外泌体比例对胰腺癌具有良好的诊断效能，AUC可达0.8以上，优于传统CA19-9。

五、纳米药物与纳米医药表征

纳米药物是纳米流式另一个重要应用领域。随着脂质纳米颗粒（LNP）mRNA疫苗、聚合物纳米药物、脂质体药物等产品的上市，纳米药物质量表征变得越来越重要。

5.1 纳米药物表征需求

纳米药物需要表征的关键质量属性包括：

  • 粒径分布：纳米药物粒径显著影响体内分布和药效
  • 颗粒浓度：确定单位剂量中的颗粒数
  • 载药率：多少颗粒携带了药物
  • 包封率：药物是否成功包封
  • 表面修饰：PEG化、靶向配体修饰效率

传统检测方法很难获得单颗粒水平的这些信息，纳米流式可以提供独特的视角。

5.2 主要应用

1. 粒径分布测量

  • 纳米流式可以快速测量数千到数万个颗粒，得到准确的粒径分布
  • 相比DLS，对多分散样品分辨率更好

2. 载药率检测

  • 通过荧光标记药物，可以检测每个纳米颗粒是否携带药物
  • 统计得到载药率，这是其他方法很难做到的
  • 优化制备工艺，提高载药率

3. 靶向配体修饰表征

  • 荧光标记靶向配体，检测每个纳米颗粒的配体修饰水平
  • 分析修饰效率，优化修饰工艺

4. 稳定性检测

  • 储存过程中纳米颗粒是否聚集
  • 不同条件下粒径分布变化
  • 评价制剂稳定性

案例：mRNA-LNP疫苗表征，纳米流式可以：

  • 检测LNP粒径分布，通常LNP粒径在50-150 nm
  • 确认mRNA包封率，利用荧光标记mRNA
  • 检测制备过程中是否形成聚集
  • 保证疫苗批次一致性

5. 病毒检测与疫苗研发

病毒颗粒尺寸正好落在纳米流式检测范围内：

  • 大多数病毒直径在20-300 nm
  • 纳米流式可以计数病毒颗粒浓度
  • 可以检测病毒表面蛋白表达
  • 监测疫苗生产过程中的病毒浓度
  • 鉴别完整病毒和空颗粒，提高疫苗质量

例如，mRNA疫苗生产过程中，需要检测LNP颗粒浓度，纳米流式可以快速准确给出结果，比电子显微镜方便得多。

六、其他应用领域

6.1 细菌和微生物检测

  • 小型细菌（如支原体）直径通常<500 nm，传统流式检测困难
  • 纳米流式可以有效检测小型细菌
  • 可以用荧光标记特异性抗体，鉴定病原菌
  • 检测血液、尿液中的低浓度病原菌，快速诊断感染

6.2 细胞凋亡与微囊泡研究

  • 细胞凋亡过程中会产生凋亡小体，尺寸通常在几百纳米
  • 纳米流式可以计数凋亡小体，定量检测细胞凋亡水平
  • 研究微囊泡释放的生物学功能

6.3 血小板微颗粒检测

  • 血小板活化后释放血小板微颗粒（PMP），尺寸~100-500 nm
  • PMP参与血栓形成、免疫调节
  • 纳米流式可以准确计数PMP，用于血栓性疾病研究

6.4 单细胞分泌囊泡分析

  • 研究不同细胞亚群分泌的EVs的差异
  • 分析单细胞水平的分泌异质性
  • 发现肿瘤细胞和正常细胞分泌EVs的差异

七、纳米流式技术最新进展

7.1 超高灵敏度检测

近年来，纳米流式灵敏度进一步提高，现在顶尖仪器可以检测20 nm的颗粒，对于荧光标记的生物颗粒，检测下限可以到~30 nm，几乎覆盖了所有感兴趣的生物纳米颗粒。

7.2 多光谱纳米流式

现代纳米流式已经支持4-6色荧光同时检测，可以对同一个纳米颗粒进行多参数标记分析，获得更丰富的信息，类似传统流式的多色分析。

7.3 绝对计数

  • 新一代纳米流式可以准确进行绝对颗粒计数
  • 不需要校准微球，或者简单校准即可
  • 临床诊断需要准确的浓度信息，绝对计数非常重要

7.4 微流控纳米流式

研究人员也在探索将纳米流式和微流控芯片结合，进一步减小仪器体积，降低样本消耗，适合床旁检测应用。

7.5 荧光纳米颗粒编码

  • 利用不同荧光强度编码不同微球，用于液相芯片
  • 纳米级编码微球，提高芯片密度
  • 纳米流式可以解码纳米编码微球，实现高通量多重生物检测

八、优势与局限性

8.1 纳米流式的优势

相比其他纳米颗粒表征方法，纳米流式的优势：

  1. 高通量：每秒检测数千个颗粒，一小时可以分析数百万颗粒，这是电镜和NTA无法比拟的
  2. 单颗粒多参数：每个颗粒同时获得散射（粒径）和多色荧光信息
  3. 定量准确：可以绝对计数颗粒浓度，重复性好
  4. 直接检测：对于体液样本可以直接检测，不需要超速离心纯化，减少样品损失和偏差
  5. 成本适中：商品化仪器成本比超分辨显微镜低，维护成本也不高

8.2 现有局限性

纳米流式也存在一些局限性：

  1. 需要荧光标记：对于无标记检测灵敏度有限，大多数应用需要荧光标记抗体
  2. 尺寸校准：散射强度和粒径的关系需要校准，不同材质（聚苯乙烯vs生物颗粒）折射率不同，需要不同校准曲线
  3. 极低浓度检测：当颗粒浓度非常低时，检测时间较长
  4. 分辨率：对于粒径非常接近的颗粒，分辨率不如电子显微镜

8.3 小结

总体来看，纳米流式填补了传统流式和电镜/NTA之间的技术空白，提供了一种独特的高通量定量分析纳米颗粒的方法，在许多领域已经不可替代。

九、总结与展望

纳米流式检测技术经过二十多年的发展，已经从实验室的探索性技术发展成为一个成熟的商业化技术，在细胞外囊泡研究、纳米药物表征、病毒检测、临床诊断等领域得到广泛应用。

未来发展方向可以归纳为几个方面：

1. 灵敏度进一步提高：有望将检测下限进一步降低到10 nm，覆盖更小的生物颗粒

2. 更高通量多色检测：增加荧光通道数量，支持更多参数同时检测

3. 小型化和便携化：开发小型化纳米流式仪器，适合临床床边检测和资源有限地区使用

4. 单细胞EVs组学：结合分选技术，实现对特定EVs亚群分选后进行组学分析，获得更深层次的生物学信息

5. 临床转化：开发标准化的临床检测试剂盒，用于癌症诊断、预后监测，造福患者

纳米流式技术作为一个新兴的交叉学科领域，融合了光学、流体力学、电子学、生物学，未来还有很大发展空间，必将在纳米生物医学领域发挥越来越重要的作用。
