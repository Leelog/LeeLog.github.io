CD19 CAR-T治疗B细胞淋巴瘤：耐药机制与应对策略

引言

CD19是CAR-T最成功，最经典的靶点，没有之一。从第一款获批CAR-T就是针对CD19，到现在已经有近10款产品获批，绝大多数都是针对CD19，在B细胞非霍奇金淋巴瘤、急性淋巴细胞白血病都取得了非常惊艳的疗效，让很多复发难治患者获得了长期生存甚至治愈可能。

但是，和所有抗肿瘤治疗一样，耐药也不可避免。临床上，大概有30-40%患者在接受CD19 CAR-T治疗后达到缓解之后还是会复发，其中很大一部分复发伴随CD19抗原丢失，这对临床来说是个很大挑战。

近年来，随着对耐药机制研究深入，我们对CD19 CAR-T耐药有了更清晰认识，也开发出很多应对策略，本文系统梳理一下CD19 CAR-T耐药机制，以及临床上现在常用应对策略，给大家一个全面参考。

一、CD19分子生物学功能与CAR-T识别机制

1.1 CD19是什么？

CD19全称分化簇19，是一个跨膜糖蛋白，分子量大约95kD，表达于几乎所有B细胞谱系，从 pro-B细胞到成熟B细胞都表达，浆细胞不表达，正好淋巴瘤白血病都是B细胞来源，所以是非常理想靶点。

功能方面：CD19是B细胞受体（BCR）共受体，能够调节BCR信号强度，促进B细胞活化增殖，对B细胞发育生长分化非常重要，但是CD19敲除小鼠模型显示，CD19不是必须，所以作为治疗靶点，敲除B细胞不影响正常造血干细胞分化，安全性有保证。

1.2 CAR-T为什么选择CD19？

  • 几乎所有B细胞肿瘤都表达：大多数B-ALL、DLBCL、FL、MCL都表达CD19，阳性率超过95%
  • B细胞之外正常组织不表达：造血干细胞不表达，其他组织不表达，off-tumor毒性低
  • 细胞表面暴露，容易被CAR-T识别：胞外段暴露，scFv容易结合

基于以上优点，CD19顺理成章成为CAR-T第一个成功靶点，一直到现在还是最常用靶点。

二、CD19 CAR-T治疗耐药流行病学

临床上，复发耐药大致可以分为两类：

复发类型 占比 说明
CD19阳性复发 ~40-50% 保留CD19表达，肿瘤细胞还在，但是CAR-T消失了，可能是CAR-T持久性差，或者肿瘤微环境抑制
CD19阴性复发 ~50-60% CD19抗原丢失，CAR-T找不到靶点了

总体来看，一线治疗后复发难治DLBCL，接受CD19 CAR-T治疗后，**二年无进展生存率大约 40-50%**，也就是说差不多一半左右会复发，耐药还是比较常见的。

三、CD19 CAR-T耐药分子机制

3.1 抗原依赖性耐药：CD19抗原丢失

这是最常见，最经典耐药机制，又可以分为几种不同分子机制：

3.1.1 splicing变异 选择性剪切产生缺失CD19胞外段

肿瘤细胞通过选择性剪切，改变CD19蛋白剪切方式，产生变异，保留胞内段，丢掉胞外段，这样CAR-T结合位点没了，就不能识别了，这种方式占CD19阴性复发大概一半左右。

3.1.2 完全基因缺失

肿瘤细胞通过染色体缺失，完全丢掉CD19基因位点，这种也会导致CD19完全表达缺失，常见较大缺失，涉及范围比较广。

3.1.3 表型转换

比如原来B细胞淋巴瘤，治疗后转变成了组织细胞淋巴瘤或者骨髓系白血病，这样自然就不表达CD19了，这种是肿瘤细胞谱系转换，也会导致抗原丢失。

3.1.4 抗原下调

没有完全丢失，表达量下调，低于检测阈值，CAR-T不能有效识别结合，也会导致耐药，这种叫做抗原密度降低，低于阈值就逃逸了。

3.2 抗原非依赖性耐药机制

即使CD19保留表达，也可能耐药，主要机制包括：

3.2.1 CAR-T细胞增殖持久性差

转进去之后，CAR-T细胞在体内不能有效增殖，很快就被清除了，肿瘤细胞还在，CAR-T没了，自然就复发了，可能和T细胞质量有关，患者之前接受过多轮化疗，T细胞质量比较差。

3.2.2 T细胞耗竭

持续抗原刺激导致T细胞耗竭，表达耗竭标记PD-1、TIM-3、LAG-3，效应功能降低，不能有效杀伤肿瘤。

3.2.3 肿瘤微环境抑制

肿瘤微环境有很多抑制性免疫细胞，比如Treg、M2型巨噬细胞，还有抑制性细胞因子比如TGF-β、IL-10，抑制CAR-T细胞浸润活化，不能有效进入肿瘤内部杀伤肿瘤。

3.2.4 抗原丢失克隆选择

肿瘤治疗之前就存在少量CD19阴性克隆，治疗之后阴性克隆被选择出来优势生长，导致复发。

四、临床上应对CD19 CAR-T耐药策略

现在临床上已经开发出很多有效策略应对耐药，我们逐个梳理：

4.1 双靶点CAR-T：CD19+CD22 双靶点

既然 CD19 容易丢，那我们同时靶向CD19+CD22，两个都在B细胞表达，同时阻断，几乎不可能同时丢掉两个，这样能降低复发概率，现在临床上已经广泛应用，很多研究显示，双靶点能降低CD19阴性复发率，提高PFS。

现在开发双靶点产品很多：

  • 国内公司很多在做，比如亘喜生物、科济药业都有管线
  • 临床上对于高危患者，一线直接用双靶点已经成为趋势

4.2 BCMA CAR-T治疗浆细胞肿瘤其实也是一样，现在已经有 CD26/BCMA 双靶点，也是类似思路，降低 BCMA 耐药复发，和B细胞淋巴瘤一样。

4. 靶向不同抗原序贯治疗

如果复发之后CD19阴性，可以换其他B细胞抗原CAR-T，比如CD22 CAR-T，现在有数据显示，再次接受CAR-T治疗还是能获得不错缓解，大概一半左右患者能再次获得CR，适合复发后挽救治疗。

4. 新型抗原：CD79b、BAFF-R 等新型靶点

针对CD19治疗后复发，现在也开发出新型B细胞抗原，比如：

  • CD79b：也是B细胞特异性表面蛋白，信号传导相关
  • BAFF-R：B细胞活化受体，表达于B细胞
  • FCRL5： also B cell specific

这些新型靶点，CD19复发后依然表达，所以可以作为CAR-T靶点，现在临床上已经有临床试验在做，初步结果不错，给CD19复发患者提供新选择。

4. 双特异性抗体

现在已经有CD19/CD3 BiTE，比如blinatumomab，对于CD19 CAR-T复发后，也可以用双特异性抗体治疗，不需要基因改造，直接给药，也能诱导内源性T细胞杀伤CD19阳性肿瘤，给患者另一个选择。

4. 自体造血干细胞移植联合巩固

如果 CAR-T 治疗后获得缓解，接受自体造血干细胞移植巩固能够提高持续缓解时间，减少复发，适合年轻 fit 患者。

4. 免疫检查点抑制剂解救

如果CAR-T后复发，而且T细胞耗竭，表达PD-1，用PD-1/PD-L1抑制剂能够逆转耗竭，恢复CAR-T功能，现在临床上也有尝试，联合治疗。

五、耐药监测 minimal residual disease 检测

治疗后监测MRD非常重要，能够早期发现复发，及时干预，现在常用方法：

  • 流式细胞学：检测CD19阳性细胞
  • 二代测序：检测肿瘤细胞克隆，看看有没有CD19变异
  • PET-CT：影像学评估肿块

定期监测能够早期发现复发，早点干预，提高二次挽救治疗成功率。

六、不同复发风险分层预后因素分析

临床上，我们可以治疗前预测复发风险，对于高复发风险患者，提前干预，提高治愈率：

高复发风险因素：

  • LDH升高：治疗前LDH显著升高提示肿瘤负荷大，复发风险高
  • 肿块 > 10 cm：大肿块肿瘤细胞数量多，更容易耐药复发
  • 三线以上治疗：多线治疗后T细胞质量差，CAR-T扩增差，更容易复发
  • 骨髓受累：骨髓累及提示肿瘤负荷大，扩散广泛，复发风险高
  • TP53突变/17p缺失：遗传学高危，对化疗耐药，也容易对CAR-T耐药

临床医生可以根据这些因素，对患者进行风险分层，低危患者可以单用CD19 CAR-T，高危患者考虑一线直接用双靶点CAR-T，或者联合其他治疗，降低复发风险，提高疗效。

七、未来发展方向：预防大于治疗

现在越来越多研究聚焦在预防耐药，而不是等到耐药再挽救：

7.1 一线双靶点CAR-T

现在越来越多临床研究显示，一线直接用CD19+CD22双靶点CAR-T，能够降低CD19阴性复发率，提高PFS，比单靶点改善预后，对于高危患者，一线直接双靶点已经成为趋势，未来会越来越普及。

7.2 联合靶向/免疫治疗

CAR-T联合BTK抑制剂（对于MCL），或者免疫检查点抑制剂，能够改善肿瘤微环境，提高CAR-T疗效，降低复发风险。

7.3 可控开关CAR-T

新一代CAR-T设计加入可控开关，万一出现毒性或者耐药，可以关闭CAR表达，提高安全性，也能控制耐药克隆。

八、案例分享：临床上一例 CD19 CAR-T 复发后成功挽救

我们看一个真实案例：

患者，男性，56岁，复发难治弥漫大B细胞淋巴瘤，一线R-CHOP方案后复发，二线化疗后再次复发，给予CD19 CAR-T治疗，获得完全缓解，18个月后复发，活检证实CD19阴性，给予CD22 CAR-T挽救治疗，再次获得完全缓解，现在已经随访24个月，持续缓解中。

这个案例说明，即使CD19耐药，现在还有很多方法可以挽救，患者仍然能够获得长期生存，不需要放弃。

九、总结与展望

CD19 CAR-T已经革命性改变了复发难治B细胞淋巴瘤治疗，让很多患者获得长期生存，耐药不可避免，但是现在我们对耐药机制已经有比较清晰认识，临床上也开发出很多有效应对策略，包括双靶点CAR-T、新型靶点CAR-T、序贯治疗、双特异性抗体等等，让耐药患者仍然有药可医，能够获得再次缓解。

未来，随着技术进步，预防耐药策略越来越先进，比如一线直接用双靶点，能够降低耐药概率，提高治愈率，给患者带来更好疗效。
